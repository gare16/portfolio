Pada proyek ini, saya mengembangkan aplikasi web internal untuk kebutuhan pelaporan data di SDI Bappenas. Aplikasi ini dibangun dengan fitur utama seperti sistem autentikasi berbasis token JWT untuk menjaga keamanan akses pengguna.

![SDI Bappenas](/portfolio/loginsdi.png "SDI Bappenas")

Saya juga mengimplementasikan modul CRUD untuk memungkinkan pengguna membuat, mengedit, dan menghapus laporan dengan tampilan antarmuka yang user-friendly. Selain itu, saya menambahkan fitur ekspor data ke dalam format CSV dan visualisasi laporan menggunakan grafik interaktif berbasis Chart.js.

![SDI Bappenas](/portfolio/monitoring.png "Monitoring Section")

Aplikasi ini juga dilengkapi dengan sistem pembuatan surat tugas otomatis yang terhubung langsung dengan data laporan. Selama pengembangan, saya menerapkan prinsip responsiveness dan accessibility agar aplikasi dapat diakses optimal dari berbagai jenis perangkat.

![SDI Bappenas](/portfolio/letter.png "Letter Section")

Proyek ini saya kerjakan dengan kolaborasi tim menggunakan Git untuk version control dan Figma sebagai alat desain UI/UX.
